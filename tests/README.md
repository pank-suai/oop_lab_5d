# Директория тестов для Лабораторной работы №5Д

## Структура директории

```
tests/
├── test_triangle.cpp       - Модульные тесты класса Triangle
├── test_integration.cpp    - Интеграционные тесты всей программы
├── test_memory_leak.cpp    - Тест на утечки памяти
├── test_move_debug.cpp     - Отладочный тест для метода move()
└── TEST_REPORT.txt         - Полный отчет о тестировании
```

## Компиляция и запуск тестов

### Модульные тесты

```bash
g++ -std=c++23 -Wall -Wextra -g tests/test_triangle.cpp -o build/test_triangle
./build/test_triangle
```

Результат: 14/14 тестов пройдено

### Интеграционные тесты

```bash
g++ -std=c++23 -Wall -Wextra -g tests/test_integration.cpp -o build/test_integration
./build/test_integration
```

Результат: 11/11 тестов пройдено

### Тест утечек памяти

```bash
g++ -std=c++23 -Wall -Wextra -g tests/test_memory_leak.cpp -o build/test_memory_leak
echo "" | ./build/test_memory_leak
```

Результат: 10000 треугольников создано/удалено без утечек
