## Структура класса Triangle

```cpp
class Triangle : public Shape, public Rotatable, public reflectable
{
protected:
  Line* _side1;  // Первая сторона треугольника
  Line* _side2;  // Вторая сторона треугольника
  Line* _side3;  // Третья сторона треугольника
  
public:
  // Конструктор: три вершины треугольника
  Triangle(const Point& p1, const Point& p2, const Point& p3);
  
  // Методы от Shape
  void draw(Screen* screen) const override;
  void move(Point p) override;
  Point getLeftTop() const override;
  Point getRightTop() const override;
  Point getLeftBottom() const override;
  Point getRightBottom() const override;
  
  // Методы от Rotatable (заглушки)
  void rotateLeft() override;
  void rotateRight() override;
  
  // Методы от reflectable (заглушки)
  void flipHorisontally() override;
  void flipVertically() override;
  
  // Деструктор
  ~Triangle();
};
```

## Ключевые моменты реализации

1. **Множественное наследование** - Triangle наследуется от трех классов одновременно
2. **Виртуальные функции** - все методы помечены как `virtual override`
3. **Управление памятью** - деструктор удаляет все три линии
4. **Габаритные точки** - используются `std::min` и `std::max`
5. **Композиция** - треугольник состоит из трех линий
